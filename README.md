# Base Api

Модуль предназначен для создания единой точки входа при создании клиентского API.

# Установка

Выполнить
```
yarn add @garpix/base-api
```

## Пример использования

```js
import BaseApi from '@garpix/base_api';

class Api extends BaseApi {

}
```
